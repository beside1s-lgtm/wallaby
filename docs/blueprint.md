# **App Name**: 체육 성장 기록 시스템

## Core Features:

- 교사 로그인: 교사 로그인 버튼만으로 로그인
- 학생 로그인: 학년, 반, 번호, 이름 입력 후 로그인
- 교사 대시보드 - 학생 관리: 학생 개별/CSV 일괄 등록 및 정보 삭제
- 교사 대시보드 - 측정 종목 관리: 측정 종목 추가 및 삭제
- 교사 대시보드 - 성과 분석: 학생 기록 조회, 개인/전체 비교 그래프, CSV 내보내기, 이름 검색
- 교사 AI 도우미: AI가 학생 장단점 분석 및 훈련 방법 제안 (tool)
- 학생 대시보드 - 결과 기록: 측정 종목 선택 후 결과 입력 (하루 1회)
- 학생 AI 피드백: AI가 운동 수행 결과 기반 피드백 제공 (tool)
- 학생 대시보드 - 진행 상황 추적: 측정 결과 그래프 표시 (종목별 선택 가능)
- 로그아웃 버튼: 우측 상단 로그아웃 버튼

## Style Guidelines:

- 주요 색상: 안정감과 신뢰감을 주는 짙은 파란색 (#3F51B5).
- 배경 색상: 깨끗하고 차분한 느낌의 밝은 파란색 (#E3F2FD).
- 강조 색상: 시선을 끄는 주황색 (#FF9800)을 인터랙티브 요소에 사용.
- 본문 및 제목 글꼴: 현대적이고 가독성이 좋은 'PT Sans'.
- 간단하고 명확한 탐색 및 작업 아이콘.
- 명확한 데이터 프레젠테이션을 제공하는 깨끗하고 미니멀한 레이아웃.
- 부드러운 사용자 경험을 위한 미묘한 전환 및 피드백 애니메이션.
