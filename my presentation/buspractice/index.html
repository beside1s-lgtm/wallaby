📍 1. 담당 버스 확인 및 선택

선생님 페이지에 로그인하면 가장 먼저 본인에게 배정된 담당 버스를 확인해야 합니다. 상단 필터에서 버스 호차와 날짜, 노선 타입을 선택하면 해당 운행에 맞는 학생 명단과 좌석표가 자동으로 불러와집니다. 아래 필터를 조작해 보세요.

🛡️ 담당 버스 보안

로그인한 계정에 할당된 버스만 목록에 나타납니다. 다른 버스를 운행해야 하는 특이 사항 발생 시 관리자에게 즉시 문의하여 권한을 조정받으세요.

⚡ 빠른 전환

등교/하교/방과후 탭을 클릭하면 별도의 페이지 이동 없이 즉시 해당 시간대의 학생 명단으로 화면이 갱신됩니다.

💺 2. 좌석표 및 실시간 탑승 관리

실제 버스 구조를 본떠 만든 좌석표(Seat Map)를 통해 학생들의 탑승 여부를 체크합니다. 좌클릭은 상태 변경, 우클릭은 자리 교체 기능을 수행합니다. 아래 시뮬레이터에서 직접 조작해 보세요.
